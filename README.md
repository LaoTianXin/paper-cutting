# 🎭 AI 剪纸互动体验 - 正定传统剪纸艺术

这是一个基于 React + TypeScript + MediaPipe 的 AI 剪纸风格化红色艺术互动平台，专门为正定传统剪纸文化设计。

## 🌟 项目简介

本项目旨在通过 AI 技术，将游客的实景照片提取并转化为具有“正定传统剪纸”风格的艺术形象。系统支持手势互动，为观众提供沉浸式的数字文化体验。

### 硬件支持
- **终端设备**：立式竖屏一体机（竖屏显示，适配 9:16 或更高比例）
- **捕捉设备**：体感摄像头（精准捕捉观众姿态与手势）

## ✨ 功能特点

### 1. 🎨 剪纸风格化
- **艺术风格**：深度锁定“正定传统剪纸”艺术特色。
- **视觉特征**：线条粗细均匀、以中国红为主色调、融入正定特色的纹样边角装饰。
- **AI 重绘**：基于观众照片，通过 AI 算法生成高度风格化的剪纸形象。

### 2. 👋 智能交互流程
- **全流程手势控制**：从扫描启动到拍照确认，无需物理接触。
- **状态机引导**：
  1.  **扫描启动**：自动检测观众进入范围。
  2.  **手势确认**：检测特定手势启动倒计时。
  3.  **倒计时拍照**：视觉反馈引导拍照。
  4.  **AI 风格化处理**：后台处理并渲染生成剪纸效果。
  5.  **成品展示**：展示最终生成的红色剪纸艺术品。

### 3. 📱 竖屏优化
- **UI 设计**：专为竖屏一体机定制化设计，采用大字号、鲜明色彩和流畅动画。
- **响应式适配**：支持主流垂直屏幕分辨率（如 720x1280, 1440x3200 等）。

## 🚀 快速开始

### 环境要求
- Node.js 16+
- pnpm（推荐）或 npm

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm dev
```
访问 http://localhost:5173

### 构建生产版本
```bash
pnpm build
```

## 📚 技术栈

### 核心技术
- **React 18**: UI 框架
- **TypeScript**: 类型安全
- **Vite**: 构建工具
- **MediaPipe**: 手势与姿态识别
- **Canvas API**: 图像采集与实时渲染
- **Tailwind CSS**: 界面样式布局
